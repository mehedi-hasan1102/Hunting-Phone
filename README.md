# Hunting Phone - Phone Store Website

## Overview
**Hunting Phone** is a **frontend website for a phone store** built using **HTML, CSS, JavaScript**, and **Open API integration**. This project demonstrates fetching and displaying phone data dynamically from an API, providing a responsive and interactive shopping interface.  

The focus is on frontend development, API usage, and responsive design.

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Open API for phone data
- Responsive Design

## Features
- Dynamic phone listing using API data
- Responsive and mobile-friendly layout
- Clean and modern UI
- Interactive search and filter functionality
